# NOIR.co — Modern Clothing E-Commerce Website

**NOIR.co** is a premium, minimalist clothing brand website built with pure HTML, CSS, and JavaScript. It offers a sleek, dark-themed shopping experience designed for the modern individual who values simplicity and style.

## ✨ Features

- **Modern Dark Theme** — Clean teal & navy color palette with premium aesthetics
- **Product Catalog** — 11 curated clothing items across Tops, Bottoms, Outerwear & Accessories
- **Interactive Product Modal** — Click any product to view details, select sizes, and add to cart
- **Image Toggle** — Switch between model photos and clothing-only flat-lay views in the modal
- **Shopping Cart** — Fully functional cart with quantity controls, saved to localStorage
- **Product Search & Filter** — Search by name and filter by category on the Shop page
- **Smooth Animations** — Scroll-reveal effects, hover transitions, and micro-interactions
- **Responsive Design** — Fully responsive across desktop, tablet, and mobile devices
- **Infinite Marquee** — Auto-scrolling product showcase banner

## 🛠️ Tech Stack

- **HTML5** — Semantic structure with SEO meta tags
- **CSS3** — Custom properties (variables), Grid, Flexbox, animations
- **Vanilla JavaScript** — Cart system, modals, filtering, localStorage
- **Google Fonts** — Poppins typeface
- **Material Icons** — Google Material Icons Outlined

## 📁 Pages

| Page | Description |
